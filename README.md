# Project Vagabond

A travel community for users to share city-specific tips ("posts" or "logs") about their favorite locations around the world. Full-stack Rails App deployed on Heroku (heroku link).

### Technologies Used

* Ruby, Rails
* Paperclip
* Materialize, SCSS
* AngularJS

### Deliverables

* Basic Auth & Profiles
* Full CRUD
* Validations & Authorizations
* Commenting
